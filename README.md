# DevCrew_s 日本語版（AI社員 / AIエージェント仕様テンプレート）

## 概要
本リポジトリは、DevCrew_s を土台にした  
日本企業向け AI社員 / AIエージェント仕様テンプレート群である。

単なる翻訳ではなく、日本の業務運用に適合する形へ再設計することを目的とする。

---

## このプロジェクトの目的
- 日本企業でそのまま使える AI社員仕様テンプレートを作る
- 業務設計・運用設計まで含めた実務資産を整備する
- GitHub公開・note販売・実務活用につながる形にする

---

## 本家 DevCrew_s との関係
- 本家は 01_original_source に参照原本として保持している
- 日本版は 02_jp_specs で別管理する
- 本家の構造・思想は維持しつつ、日本向けに再設計する
- 差分は明示的に管理する

---

## 日本版で重視する要素
- 敬語 / 対人配慮
- 稟議 / 決裁
- 承認フロー
- 責任分界
- 監査証跡
- 個人情報 / 機微情報管理
- 委託先 / 外部連携管理
- 引き継ぎ前提の記録設計

---

## リポジトリ構造
01_original_source/   # 本家参照原本（編集しない）
02_jp_specs/          # 日本語版仕様（編集対象）
03_decisions/         # 意思決定ログ
04_changelog/         # 差分履歴
05_templates/         # 再利用テンプレート
06_logs/              # 作業ログ
07_assets/            # 補助資料
08_exports/           # 出力物
99_archive/           # アーカイブ

---

## 作業ルール（重要）
- 作業のメインはローカル
- GitHubは保存先（正本）
- 変更は都度コミット・プッシュする
- 本家は直接編集しない
- 日本版のみを編集対象とする
