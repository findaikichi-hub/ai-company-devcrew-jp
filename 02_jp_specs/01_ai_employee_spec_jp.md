# 01_ai_employee_spec_jp

## 位置づけ
本ファイルは、日本企業向け AI社員 / AIエージェント仕様テンプレートの中核ドラフトとする。

## このテンプレートの目的
- 日本企業で使いやすい AI社員の仕様雛形を定義する
- 本家の設計思想を維持しながら、日本独自要件を追加する
- 各業務別テンプレートの共通土台にする

## 本家から継承する要素
- 役割定義
- 目的 / 責務
- 思考様式
- 記憶の扱い
- 使用可能ツール
- 通信プロトコル
- ガバナンス / 倫理 / 安全
- 運用 / ライフサイクル管理
- ログ / 監査 / 評価 / 改善

## 日本版で追加する要素
- 敬語と対人配慮
- 稟議 / 承認フロー
- 責任分界
- 監査証跡
- 個人情報 / 機微情報の扱い
- 委託先 / 外部連携管理
- 引き継ぎ前提の記録設計

## 今後ここに詰める項目
- 役割定義
- 入出力
- 判断基準
- エスカレーション条件
- 記録ルール
- 監査ログ要件
- 失敗時の扱い
- 改善ループ
